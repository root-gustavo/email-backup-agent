# email-backup-agent

Agente em segundo plano para backup automático de arquivos de e-mail.

O **email-backup-agent** é um utilitário que roda na inicialização do sistema, identifica arquivos de dados de e-mail (como PST), permite a seleção inicial dos arquivos a serem acompanhados e realiza cópias compactadas para sincronização com o SharePoint.

O projeto foi pensado para funcionar de forma discreta, automatizada e confiável, servindo como base para estudos e evolução em automação, manipulação de arquivos, empacotamento com PyInstaller e integração com APIs.

> Este README será expandido conforme o projeto evoluir.
